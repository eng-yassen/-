# -
قمت بتصميم صفحة تعريفية شخصية (Portfolio) لعرض مهاراتي ومشاريعي بشكل احترافي. الصفحة تعرض نبذة عني، المهارات الأساسية، وأهم المشاريع التي نفذتها باستخدام HTML و CSS فقط. التصميم متجاوب مع جميع الشاشات (موبايل – تابلت – كمبيوتر) ويتميز بتنظيم الكود وسهولة التعديل. يمكن استخدام الصفحة كبورتفوليو لمبرمج أو فريلانسر يرغب بعرض أعماله بطريقة جذابة وبسيطة
